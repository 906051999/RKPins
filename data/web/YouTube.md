# 广告机制
- 视频播放前、中、后插入广告
- 页面横幅广告
- 创作者可选择是否在视频中插入广告

# 账号机制
- 不登录可观看大部分视频
- 注册免费，可订阅频道、点赞评论
- 创作者可上传视频、直播

# 付费机制
- YouTube Premium会员可去广告、观看独家内容
- 部分频道提供会员制
- 超级留言等打赏功能

# 特点
- 海量视频内容
- 强大的推荐算法
- 支持多种分辨率和播放速度
- 提供直播功能
- 创作者可通过广告分成获得收入
